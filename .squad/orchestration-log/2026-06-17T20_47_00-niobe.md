### 2. Script Capabilities

Scripts cover all route inspection commands with prompt-driven and non-interactive (flag/env-var) modes:

- `gcloud compute routes list --project <project> --filter "network:<network>"`
- `gcloud compute routers get-status <router> --region <region> --project <project>`
- `gcloud compute routers describe <router> --region <region> --project <project>`
- `gcloud compute vpn-tunnels describe <tunnel> --region <region> --project <project>`
- `gcloud compute vpn-tunnels list --project <project> --regions <region>`
- `gcloud compute forwarding-rules list --project <project> --regions <region> --filter "target:targetVpnGateways/<gateway>"`
- `gcloud compute routes describe <route> --project <project>`

### 3. Default Values

All defaults sourced from `terraform/gcp/` resource names:
- Region: `us-central1`
- VPC network: `vpnlab-vpc`
- Cloud Router: `vpnlab-router` (optional, graceful when disabled)
- Classic VPN gateway: `onpremvpn`
- VPN tunnel: `vpn-to-azure`
- VPN static route: `vpn-to-azure-route-1`

Cloud Router probing is optional and graceful because `enable_interconnect=false` by default.
